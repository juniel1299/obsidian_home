# 자바스크립트의 마우스이벤트
1. onmouseover/onmouseenter 
	- 해당 객체 영역에 마우스 커서가 진입하는 순간 발생 
2. onmouseout/onmouseleave
	- 해당 객체 영역에 마우스 커서가 빠져나가는 순간 발생
3. onmousedown
	- 해당 객체 영역에 마우스 버튼을 누르는 순간 발생
4. onmouseup
	- 해당 객체 영역에 마우스 버튼을 떼는 순간 발생
5. onmousemove 
	- 해당 객체 영역에서 마우스를 움직일 때 발생
