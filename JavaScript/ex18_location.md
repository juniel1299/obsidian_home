# 자바스크립트의 location
- window > location 
- 현재 창의 페이지 주소 참조 객체 > URL 조작
- console.log(location.href); // 읽기

**페이지 이동 (자바스크립트)**

- location.href = 'ex16_window.html'; //쓰기 
- location.reload(); //새로고침 
- location.assign('ex16_window.html');
