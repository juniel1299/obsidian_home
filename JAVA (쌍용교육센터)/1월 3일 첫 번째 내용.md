#  연산자  Variable

>10 + 20 = 30을 만들려고 함 
>A + B  > 이항 연산자 (피연산자 2개)
>연산자 우선순위는 사칙연산 개념과 같음 > 1+2 * 3 에서 곱하기 먼저 계산하듯 
>


**첫 번째 예제**
```java
//ex 1 
System.out.println( a + "+" + b + "=" + (a + b) );
// (a + b) 를 하지 않으면 그냥 1020 이 출력이 됨 그러므로 괄호가 필요함 . 
```
## 진법
> 주민등록번호 입력 > 출력  ex) 950105; 라 할 때


**두 번째 예제 
```java
int jumin1 = 950105;

System.out.println("주민등록번호 " + jumin1);

jumin1 = 030711;

System.out.println("주민등록번호 " + jumin1);
```
