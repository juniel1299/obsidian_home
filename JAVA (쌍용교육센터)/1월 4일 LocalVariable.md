## 지역 변수 
> 여태까지 사용한 변수들을 지역 변수라고 함 
> 현재 메서드 public static void main 안에서만 쓰는 변수 > 지역 변수
>  메서드 내에서 선언한 변수 
>**변수 스코프(Scope) > 변수의 사용 영역(범위)**

> 지역 변수의 생명주기 life Cycle
> 생성 : 변수 선언문이 실행할 
> 소멸 : 변수 선언문이 포함된 블럭을 빠져나갈 때
> main 메서드 변수는 소멸이 늦음 > 메모리량 증가
