# switch 문 

역할은 if문과 거의 똑같음

## switch 문법 
switch (조건){
case 조건의 결과값: 
break;
case 조건의 결과값:
break;
}

날짜 , 윤년 계산 이런 부분에서 유리함 

```java
int lastDay = 0;

int month = 5;

//aaa: //Label > 코드의 위치를 설명하는 용도 > 표지판

  

switch (month) {

case 1:

case 3:

case 5:

case 7:

case 8:

case 10:

case 12:

lastDay = 31;

break;

case 4:

case 6:

case 9:

case 11:

lastDay = 30;

break;

case 2:

lastDay = 28;

break;

}

System.out.printf("%d은 %d일까지 있습니다.\n",month,lastDay);
```
