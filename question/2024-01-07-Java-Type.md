# Java Data Type 
자바의 자료형 (data type)에 대해 작성을 해보려고 한다.

자바에서 자료형 기준은 크게 2가지로 나뉘게 된다.
## 자바의 자료형 기준 
1. 형태
2. 길이 

이 두가지를 조건을 가지고 총 8가지의 기본형과 이외의 참조형으로 나뉘게 된다 . 

### 자료형 분류 
>  기본형, 원시형(Primitive Type), 값형(Value Type)
> 	 1. 숫자형
> 		- 정수형 (byte, short, int, long)
> 		- 실수형 (float, double)
> 		- 문자형 (char)
> 		- 논리형 (boolean)
