하나의 파일에서 모든걸 구현하지 말고 클래스를 쪼개서 구현하자 


**클래스는 일반적으로 파스칼 표기 사용** (맨 앞 글자 대문자, 단어가 여러개 합성이 된 경우 각 맨 앞 글자 대문자)

예시 

```java
class Point{
	
	//구현부
	//클래스 멤버 선언하기
	// 1. 변수 
	// 2. 메서드

	//멤버 변수
	public int x;
	public int y;
	public int z;
	public int martx;
	public int marty;

}

class Size {
	
	public String name;
	public int width;
	public int height;



}

//성적 저장 클래스
class Score{
	public String name;
	public int kor;
	public int eng;
	public int math;
}
```
