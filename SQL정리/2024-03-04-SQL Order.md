# Order by (정렬)

컬럼 순서를 정렬할 때 사용함.  

select 컬럼
from 테이블
where 조건
order by 오름/내림차순;

일 때 제일 마지막에 동작함 (4번째)  

테이블 지정 -> 조건에 맞는 데이터를 -> 컬럼에 맞춰 데이터를 가져옴-> 데이터 정렬  

```sql
select * from tblinsa order by basicpay;
```

내림차순을 하려면 order by basicpay desc; 를 해야 함
