# 블럭태그와 인라인태그의 너비 높이 정책
- width , height > 블럭태그를 위한 속성 

1. 블럭 태그의 정책
	- 기본 정책
		- 너비 : 100%(부모 영역을 꽉 채운다)
		- 높이 : 내용물의 높이가 영역의 높이가 된다

2. 명시적 기재
	- 내용물이 영역에서 벗어나면(Overflow) 영역으로 빠져나간다.
	- 너비 : 불가능
	- 높이 : 불가능
	- 예외 : 폼 태그 소속 인라인 태그 + 이미지 태그 




4. 인라인 태그 정책
	- 기본 정책
		- 너비 : 내용물의 너비가 영역의 너비가 된다.
		- 높이 : 내용물의 높이가 영역의 높이가 된다.
