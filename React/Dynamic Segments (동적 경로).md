URL Parameter : / 뒤에 아이템의 id를 명시 
-> 잘 변경되지 않는 값을 주소로 명시하기 위해 사용 

Query String : ? 뒤에 변수명과 값 명시 
-> 검색어처럼 자주 변경이 발생하는 경우에 대해 값을 주소로 명시하기 위해 사용


URL 파라미터의 경우 /:id 이렇게 명시를 해줘야함 .
