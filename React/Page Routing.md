Multi Page Application (MPA) , 서버가 여러개의 페이지를 가지고 있음 
(전통적 방식)

브라우저가 서버 사이드 렌더링을 통해 서버에 /페이지명 요청 

서버는 MPA 방식으로 요청한 페이지명에 해당하는 파일을 서버 사이드 렌더링으로 반환 
(서버 사이드 렌더링의 경우 SSR , Next.js 에서 많이 사용 )
