1번째는 data 포맷을 통해 불러오고 2번째는 import 된 변수를 통해 불러오기 떄문에 최적화에 용이함 . 

확인 방법. 

npm run build 를 터미널에 작성하면 dist 폴더 생김 -> 빌드 결과물은 npm run preview 
( npm run dev 와 비슷하지만 포트가 다르게 열림 )

해당 화면에서 개발자모드  Network 탭을 통해 확인하자.
